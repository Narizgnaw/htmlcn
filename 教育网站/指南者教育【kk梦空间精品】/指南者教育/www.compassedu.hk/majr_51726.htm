数据分析理学硕士

  谢菲尔德大学 数据分析理学硕士硕士专业介绍

  数据分析理学硕士专业介绍

培养目标：

随着当今大数据时代的到来，信息的管理和运用转化已变得至关重要。谢菲尔德大学数据分析理学硕士项目将向你展示如何将以前的学术经验运用于现实世界，并帮助你开发处理大量数据所需的技能。谢菲尔德大学数据分析理学硕士项目融合了一系列学科技术，包括计算机科学、人工智能、数学和统计，适合于计算相关学科背景的学生（例如数学、工程、物理、化学或其他物理科学）及业界在职人士。在谢菲尔德大学数据分析理学硕士项目中，你将学会分析需解释的数据类型，包括大数据集及结构化和非结构化数据。

Curriculum：

1、Scalable Machine Learning

2、Text Processing

3、Machine Learning & Adaptive Intelligence

4、Natural Language Processing

5、Industrial Team Project

6、Individual Data Analytics Dissertation

7、Information Governance and Ethics

8、Mathematical Data Science

9、Computer Security and Forensics

10、Parallel Computing with Graphical Processing Units (GPUs)

课程介绍：

1、可扩展机器学习

2、文本处理

3、机器学习和自适应智能

4、自然语言处理

5、工业团队项目

6、个人数据分析论文

7、信息治理和道德

8、数学数据科学

9、计算机安全和取证

10、并行计算和图形处理器
